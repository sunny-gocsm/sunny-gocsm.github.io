## Design language (apply by default — full version in the `gocsm-design-language` memory)
- **Hierarchy per row/item:** bold large title → quiet second line for secondary facts → clear CTA. "$ at risk" in red (`--neg-7`). Never one flat blob.
- **CTAs must look clickable:** real buttons, not faint text-links. Exactly ONE solid-blue focal action per view (`.btn-primary`); everything else triggerable uses soft-blue `.btn-accent`. Clickable rows get a hover affordance.
- **Cut low-value chrome:** greetings, redundant counts, duplicate headers. The real content is the hero.
- **Bigger, fuller:** size titles to real page-title scale; the page's primary heading is the page title (large, top), not a mid-page section header. Make sections feel full-width.

## ⚠️ Typography traps (full version in `gocsm-ds-typography-gotchas` memory) — READ BEFORE SIZING TEXT
- Inline **`font: var(--t-*)`** (size only, no family) is **invalid CSS** → silently drops to ~14px. Use **`fontSize`**.
- **`--t-h3` / `--t-h4` / `--t-h6` don't exist.** Real scale: `--t-display-xl` 34 · `--t-display-lg` 26 · `--t-heading` 20 · `--t-subheading` 16 · `--t-body-lg` 15 · `--t-body` 14 · `--t-body-sm` 13 · `--t-caption` 12 · `--t-label` 11.

## Reusable DS primitives (in `gocsm-design-system`, vendored at `src/gocsm-ds/`) — reuse, don't hand-roll
- `FixItCard` with `title` + `meta` (bold name → quiet meta; wrap dollars in a `.at-risk` span for red $) + `[data-clickable]` hover; `size="lg"` scales the title. `tone="pos"` = good-news treatment.
- `.btn-accent` — soft-blue secondary CTA (`<Button variant="ghost" className="btn-accent">`).
- `.rule-statement` — one plain-language automation rule (eyebrow → rule line → live count).
- ~60 other DS components; prefer composing them over raw markup.
